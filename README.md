# Object Oriented Programming - DT228/2
## Assignment 2 Semester 2 2017

### Overview
This game is called 'Tower Defense Ultimate'.
#### Features
- 4 Towers
- 4 Enemy types
- 4 Maps
- 3 Difficulty Modes
- Endless Mode
- Sound
- Money System
- Upgrade options
- High Scores

I used the java collections library, the minim sound library, and the processing video library.
